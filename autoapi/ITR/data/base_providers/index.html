class ITR.data.base_providers.EITargetProjector(projection_controls: ITR.configs.ProjectionControls = ProjectionControls())

Bases: EIProjector

This class projects emissions intensities from a company’s targets and historic data. Targets are specified per scope in terms of either emissions or emission intensity reduction. Interpolation between last known historic data and (a) target(s) is CAGR-based, but not entirely CAGR (beacuse zero can only be approached asymptotically and any CAGR that approaches zero in finite time must have extraordinarily steep initial drop, which is unrealistic).

Remember that pd.Series are always well-behaved with pint[] quantities. pd.DataFrame columns are well-behaved, but data across columns is not always well-behaved. We therefore make this function assume we are projecting targets for a specific company, in a specific sector. If we want to project targets for multiple sectors, we have to call it multiple times. This function doesn’t need to know what sector it’s computing for…only tha there is only one such, for however many scopes.
