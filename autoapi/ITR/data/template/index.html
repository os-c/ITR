ITR.data.template._estimated_value(y: pandas.Series) → pint.Quantity

:param y : a pd.Series that arrives via a pd.GroupBy operation. The elements of the series are all data (or np.nan) matching a metric/sub-metric.

Returns:

A Quantity which could be either the first or only element from the pd.Series, or an estimate of correct answer. The estimate is based on the mean of the non-null entries. It could be changed to output the last (most recent) value (if the inputs arrive sorted)
